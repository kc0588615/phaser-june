# Cesium UI Customization

Customize the 3D globe appearance and controls.

## Hiding Default Controls

```tsx
<Viewer
  timeline={false}
  animation={false}
  homeButton={false}
  sceneModePicker={false}
  navigationHelpButton={false}
  fullscreenButton={false}
/>
```
